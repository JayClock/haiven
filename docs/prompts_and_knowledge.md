
## Folder structure conventions
The application expects a folder structure to load prompts and knowledge from.

```
+ <root_folder>
  + prompts
  + documentation
  + <team_name_1>
    + knowledge
      + documents
        + <knowledge_base_1>.kb
        - <knowledge_base_1>.md
      + pdfs
        + <knowledge_base_2>.kb
        - <knowledge_base_2>.md
      - <markdown_knowledge_1>.md
      - <markdown_knowledge_2>.md
```
